# Week 11 — Testing demo (starter)

This is the **starting point** for the Week 11 walkthrough. It is a small,
fully working Next.js (Pages Router / JavaScript) app — a *Vehicles UI* with a
public home page, a login form, and a protected vehicles list — but it has **no
tests and no testing tools installed yet**.

Follow **`week-11/lecture/week-11-notes.md`** to add Jest (unit tests) and
Cypress (end-to-end tests) step by step. The finished result is in
**`week-11/lecture/testing-demo/my-app`** if you want to check your work.

## Run the app

```bash
npm install
npm run dev
```

Then open http://localhost:3000. Log in with **bob / myPassword**.

> **Before class:** to avoid downloading the Cypress binary live, you can run
> the `npm install --save-dev ...` commands from the walkthrough ahead of time
> so everything is cached.
